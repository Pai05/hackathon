# Backend

Express backend for Glow Research.

## Setup

1. Install dependencies:
   npm install
2. Create env file:
   copy .env.example .env
3. Start development server:
   npm run dev

## API

- GET /api/health
- POST /api/auth/request-otp
- POST /api/auth/verify-otp
- POST /api/auth/register
- POST /api/auth/login
